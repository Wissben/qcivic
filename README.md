# Install

## Windows
`xcopy qcivic.xml %APPDATA%\QtProject\qtcreator\styles`

## MacOS
`cp qcivic.xml ~/.config/QtProject/qtcreator/styles/`

## Linux
`cp qcivic.xml ~/.config/QtProject/qtcreator/styles/`
